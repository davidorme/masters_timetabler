This form shows the selected user details and permissions. You can check and uncheck options to alter permissions. New users will also have the option to approve their registration - you can set their permissions at the same time. You can also delete users, but probably check first!

  • Admin: approve users, upload and create assignments, set permissions and create new timetable courses and modules. This should be a small set of people!
  • Projects: users with this permission are allowed to create new project proposals and edit proposals.
  • Timetabler: users with this permission are allowed to edit timetable module details and manage module events.
  • Reviewer: this is not yet fully implemented but will give a user read-only access to the set of marking assignments
